Что такое «HTML»

HTML

HTML (Hypertext Markup Language) - это код, который используется для структурирования и отображения вебстраницы и её контента. Например, контент (какой-то текст) может быть представлен в виде множества параграфов, маркированных списков или с использованием изображений и таблиц данных.

Элементы HTML

  • Открывающий (< >) и Закрывающий (< />) тег
    Открывающий тег указывает, где элемент начинается или начинает действовать, закрывающий тег включает в себя косую черту перед именем элемента.
    Закрывающий элемент указывает, где элемент заканчивается.
  • Контент
    Контент элемента, который в данном случае является просто текстом.
  • Элемент Открывающий тег, закрывающий тег и контент вместе составляют элемент.

Типы HTML-элементов:

  1. Пустые элементы-Элементы HTML без содержимого. Пустые элементы не имеют конечного тега,такого как < br > < area >, < base >, < br >, < col> ,
    < embed >, < hr >, < img >, < input >, < link >, < menuitem >, < meta >, < param >,
    < source >, < track >, < wbr >;
  2. Элементы с неформатированным текстом script, style
  3. Элементы, выводящие неформатированный текст < textarea >, < title ;
  4. Элементы из другого пространства имён MathML и SVG;
  5. Обычные элементы все остальные элементы.
